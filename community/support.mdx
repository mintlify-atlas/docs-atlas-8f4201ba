### Creating a Forum Post

<Accordion title="How to write a good forum post">
  1. **Clear Title**: Summarize your issue or question
     - Good: "ESP module not rendering on 1.20.1"
     - Bad: "Help!!" or "Not working"
  
  2. **Detailed Description**:
     - What you're trying to do
     - What's happening instead
     - Steps you've already tried
  
  3. **System Information**:
     - Minecraft version
     - LiquidBounce version
     - Operating system
     - Other relevant mods
  
  4. **Logs and Screenshots**:
     - Include relevant log files
     - Add screenshots if visual issue
     - Use code blocks for logs
  
  5. **Follow Up**:
     - Reply to helpful suggestions
     - Mark as solved when resolved
     - Share the solution for others
</Accordion>

## GitHub Issues

<Warning>
  GitHub Issues are for **bug reports** and **feature requests** only. For general support, use Discord or the Forum.
</Warning>

### When to Use GitHub Issues

<CardGroup cols={2}>
  <Card title="Bug Reports" icon="bug">
    Report crashes, errors, or features not working as intended
  </Card>
  
  <Card title="Feature Requests" icon="lightbulb">
    Suggest new features or improvements to existing functionality
  </Card>
</CardGroup>

### Creating a Bug Report

<Steps>
  <Step title="Search Existing Issues">
    Check if your bug has already been reported at [GitHub Issues](https://github.com/CCBlueX/LiquidBounce/issues)
  </Step>
  
  <Step title="Use Issue Template">
    Fill out the bug report template completely with all requested information
  </Step>
  
  <Step title="Provide Reproduction Steps">
    List exact steps to reproduce the bug:
    1. Launch LiquidBounce
    2. Enable X module
    3. Perform Y action
    4. Bug occurs
  </Step>
  
  <Step title="Include Logs">
    Attach relevant log files, crash reports, and screenshots
  </Step>
  
  <Step title="Specify Environment">
    - LiquidBounce version (commit hash or build number)
    - Minecraft version
    - Operating system
    - Java version
  </Step>
</Steps>

### Creating a Feature Request

<Accordion title="Feature request template">
  **Is your feature request related to a problem?**
  Describe the problem or limitation you're experiencing.
  
  **Describe the solution you'd like**
  Clear description of what you want to happen.
  
  **Describe alternatives you've considered**
  Alternative solutions or features you've thought about.
  
  **Additional context**
  Any other context, screenshots, or examples.
</Accordion>

## How to Get Help Effectively

### Provide Complete Information

<Note>
  The more information you provide, the faster you'll get help!
</Note>

**Essential Information**:

```markdown
**Environment:**
- LiquidBounce version: [e.g., commit abc123 or build 456]
- Minecraft version: [e.g., 1.20.1]
- Fabric version: [e.g., 0.15.0]
- Operating system: [e.g., Windows 11, macOS 13, Ubuntu 22.04]
- Java version: [e.g., OpenJDK 17.0.5]

**Other mods:**
- List other mods installed (if any)

**Problem:**
- Describe what you're trying to do
- Explain what actually happens
- List what you've already tried

**Logs:**
- [Link to latest.log via mclo.gs]
- [Link to crash report if applicable]
```

### Sharing Logs

<Warning>
  Never paste entire log files directly in chat. Use a paste service!
</Warning>

**Recommended paste services**:

1. **[mclo.gs](https://mclo.gs)** - Minecraft-specific log hosting
   - Drag and drop log files
   - Automatic syntax highlighting
   - Built for Minecraft logs

2. **[Pastebin](https://pastebin.com)** - General paste service
   - Good for config files
   - Syntax highlighting available

3. **[GitHub Gist](https://gist.github.com)** - Code snippets
   - Great for scripts
   - Version control
   - Markdown support

### Finding Log Files

## Community Guidelines

### Be Respectful

- Treat everyone with respect and courtesy
- No harassment, hate speech, or discrimination
- Keep discussions civil and constructive
- Respect staff and moderator decisions

### No Piracy or Illegal Content

- Don't share cracked accounts or account generators
- No links to pirated software or content
- Respect intellectual property and licenses

### Use Appropriate Channels

- Read channel descriptions before posting
- Don't spam or cross-post across channels
- Keep off-topic discussions to appropriate channels
- Use support channels for support questions

### Don't Ask to Ask

<Warning>
  Don't ask "Can I ask a question?" - Just ask your question directly!
</Warning>

Instead of:
> "Anyone know about modules?"
> "Can someone help me?"

Do this:
> "How do I configure the Fly module to work on Hypixel? I've tried changing the mode but it's not working. [logs link]"

### Be Patient

- Wait for responses before bumping your question
- Community members help in their free time
- Complex issues may take longer to diagnose
- Check back periodically for responses
