## Modules

<AccordionGroup>
  <Accordion title="DankBobbing" icon="toggle-on">
    Dank bobbing module Adds more bobbing effect.
    
    **Usage:**
    - Toggle: `.dankbobbing`
    - Bind to key via ClickGUI settings
  </Accordion>

  <Accordion title="Derp" icon="toggle-on">
    Derp module Makes it look as if you were derping around.
    
    **Usage:**
    - Toggle: `.derp`
    - Bind to key via ClickGUI settings
  </Accordion>

  <Accordion title="HandDerp" icon="toggle-on">
    Hand derp module Switches your main hand.
    
    **Usage:**
    - Toggle: `.handderp`
    - Bind to key via ClickGUI settings
  </Accordion>

  <Accordion title="Notebot" icon="toggle-on">
    Notebot Module Automatically plays note block songs from NBS files. @author ccetl
    
    **Usage:**
    - Toggle: `.notebot`
    - Bind to key via ClickGUI settings
  </Accordion>

  <Accordion title="SkinDerp" icon="toggle-on">
    Skin Derp module Makes your skin blink (Requires multi-layer skin).
    
    **Usage:**
    - Toggle: `.skinderp`
    - Bind to key via ClickGUI settings
  </Accordion>

  <Accordion title="Twerk" icon="toggle-on">
    Twerk module Automatically sneaks and unsneaks.
    
    **Usage:**
    - Toggle: `.twerk`
    - Bind to key via ClickGUI settings
  </Accordion>

  <Accordion title="Vomit" icon="toggle-on">
    Vomit module Drops items from the inventory in a random order to make it look like the player is vomiting. If the player is in creative mode, the player will drop random block items.
    
    **Usage:**
    - Toggle: `.vomit`
    - Bind to key via ClickGUI settings
  </Accordion>

</AccordionGroup>
