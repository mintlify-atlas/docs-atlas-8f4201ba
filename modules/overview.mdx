## Quick Start

To use modules in LiquidBounce:

1. Press **Right Shift** to open the ClickGUI
2. Browse through categories
3. Click modules to toggle them on/off
4. Right-click modules to access settings

## Module Management

### Keybinds

You can bind modules to keys for quick toggling:

1. Open ClickGUI (Right Shift)
2. Right-click a module
3. Click "Bind" and press the desired key

### Configurations

Modules can be saved and loaded as configurations:

- Use `.config save <name>` to save current settings
- Use `.config load <name>` to load a configuration
- Use `.config list` to see available configs
